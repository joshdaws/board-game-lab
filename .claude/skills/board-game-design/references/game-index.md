## Archetypes Key

- **EB** = Engine Builder — Income, Tech Trees, Tableau Building, Resource Conversion
- **AC** = Asymmetric Conflict — Variable Player Powers, Area Majority, Direct Conflict
- **SP** = Spatial Puzzle — Tile Placement, Pattern Building, Spatial Constraints
- **CC** = Cooperative Crisis — Cooperative Game, Escalating Threat, Action Points
- **ES** = Economic Simulation — Market, Resource Conversion, Route/Network Building
- **NC** = Narrative Campaign — Campaign/Legacy, Character Progression, Scenario-based
- **CT** = Card Tableau — Hand Management, Multi-use Cards, Open Drafting, Combos
- **DC** = Dexterity Challenge — Flicking, Physical Skill
